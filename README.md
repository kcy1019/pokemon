# Pokemon Billiard

An OpenGL 2D practice project for fun.

### Compile and run

\*For Linux/Windows users: [GLUT](https://www.opengl.org/resources/libraries/glut/) should be installed first.

    $ make
    $ ./poke

### Controls

- Arrow keys: Adjust angle.
- Space bar: Lacunch the stick.
